# Career Navigator

Career Navigator is a Flask-based web application that helps students:

- Build Resumes
- Explore Career Paths
- Analyze Skill Gaps
- Access Learning Resources
- View Job Opportunities
- Get Career Recommendations

## Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript
