# Career Navigator

Career Navigator is a Flask-based web application designed to help students and fresh graduates prepare for their careers. The platform provides tools and resources for resume building, career guidance, skill assessment, certifications, job opportunities, and learning roadmaps.

## Features

* Resume Builder
* Career Guidance
* Skill Gap Analyzer
* Certification Resources
* Learning Roadmaps
* Latest Job Opportunities
* Learning Resources
* Contact and About Pages

## Technologies Used

* Python
* Flask
* HTML5
* CSS3
* JavaScript
* Git & GitHub

## Project Objective

The main objective of this project is to guide students in making informed career decisions, identifying skill gaps, improving employability, and accessing learning resources from a single platform.
